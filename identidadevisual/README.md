# Identidade Visual · Elofy

📂 **Suba aqui os arquivos da identidade visual da Elofy.**

Eles serão usados para aplicar o padrão oficial da marca em toda a Jornada Elofy —
especialmente nas interfaces com o cliente (página da proposta, acompanhamento da
implantação e PDF da proposta).

## O que subir

| Arquivo | Formato ideal | Para quê |
|---|---|---|
| **Manual da marca** | PDF | cores, tipografia, regras de uso do logo, tom de voz |
| **Logo principal** | SVG ou PNG (fundo transparente) | cabeçalhos e PDF |
| **Logo versão escura/clara** | SVG ou PNG | fundos claros e escuros |
| **Símbolo/ícone isolado** | SVG ou PNG | favicon, espaços pequenos |
| **Paleta de cores** | PDF/PNG ou anotação dos HEX | cores oficiais do sistema |
| **Fontes** | .woff2 / .ttf / .otf (ou nome da fonte) | tipografia oficial |
| Materiais de exemplo | PDF/PNG (apresentações, posts) | referência de estilo |

## Como subir

1. Nesta página do GitHub, clique em **Add file → Upload files**
2. Arraste todos os arquivos (pode arrastar vários de uma vez)
3. Clique em **Commit changes**

> Pode subir tudo que tiver — quanto mais material, mais fiel fica o redesign.
